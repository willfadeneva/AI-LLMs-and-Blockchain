# AI, LLM & Blockchain Builder Roadmap

A ready-to-deploy Next.js + Tailwind landing page for learning AI, LLMs, agents, blockchain, Web3, trading systems, and production software.

## What's included

- AI learning path
- roadmap.sh route links
- Free courses
- Paid courses
- University programs and course links
- YouTube channel links
- Embedded YouTube video previews
- Portfolio project ideas
- Vercel-ready Next.js setup

## Run locally

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```

## Deploy on Vercel

1. Push this project to GitHub.
2. Go to Vercel.
3. Click **Add New Project**.
4. Import the GitHub repo.
5. Deploy.

## Notes

Some YouTube embeds may fail if the video owner disables embedding. The page includes direct YouTube links as a fallback.
